# Panduan Berbagi Data

Untuk menjaga privasi dan mematuhi kebijakan platform, ikuti panduan berikut saat berbagi data hasil scraping:

1. **Simpan hanya ID**: Distribusikan berkas yang berisi `post_id` dan metadata minimal. Jangan sertakan teks mentah kecuali mendapat izin eksplisit.
2. **README pendamping**: Sertakan README yang menjelaskan bagaimana cara melakukan rehidrasi teks secara mandiri oleh penerima dengan kredensial masing-masing.
3. **Enkripsi**: Jika harus mengirim data sensitif, gunakan saluran terenkripsi dan kata sandi terpisah.
4. **Hapus data sensitif**: Singkirkan nama pengguna, tautan profil, atau informasi identitas lainnya sebelum berbagi.
5. **Catat consent**: Simpan bukti persetujuan atau izin dari pemilik konten bila diperlukan oleh kebijakan kampus atau hukum setempat.
6. **Versi dataset**: Beri penomoran versi dan tanggal pembuatan agar proses audit mudah dilakukan.
7. **Batas penggunaan**: Jelaskan batasan penggunaan data (misal hanya untuk penelitian akademik internal).

Gunakan formulir permintaan data internal departemen jika tersedia, dan konsultasikan dengan dosen pembimbing sebelum mengunggah arsip ke repositori publik.
